Overview

Demonstrating partial results of ongoing thorough studies on endometriosis segmentation, this web-based demo provides a user with the tools for segmenting endometrial implants in laparoscopic surgery videos. Following figure provides an overview of the tool's processing approach:

{% load static %}
  1. First a user provides the tool with an input video
  2. The video is analyzed using one of several provided segmentation models
  3. The analyses are utilized to additionally add a summarization bar that indicates frame-by-frame averaged segmentation confidences, while also marking the current video position.

Instructions

  1. Download the desired pre-trained models from ENID: The ITEC Endometrial Implants Dataset
  2. Extract the models into [project_folder]/django_app/media/models (create a subfolder for each model!)
  3. Start server and upload a video for analysis (please note that long videos can take a while to analyze)
  4. Wait for analysis to finish - the status can be seen by observing the server's console output but simply pressing the refresh page button () occasionally may suffice
  5. The uploaded videos should eventually be replaced by the ones containing the analysis results (the spinning will disappear)
